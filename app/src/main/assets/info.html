The table on the main screen displays a list of Braille contractions. Enter a word in the search bar to find its contracted form, or use the four Braille cells to enter a contraction and find its meaning. The table below describes the symbols used on the main screen.

Symbol Meaning
_ No adjacent letters
^ Letter(s)
^^ Word(s)

Please note that this app is not a complete guide to the use of Braille contractions. There are many rules and restrictions not stated here.

This app is intended as a quick reference tool for people using or studying Braille. It includes all of the contractions found in the 2008 UEB Braille Chart from Duxbury Systems (http://www.duxburysystems.com). I am not affiliated with Duxbury Systems.

Icons for the Info, Backspace, and Clear buttons were obtained from https://material.io/icons/.
